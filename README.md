# Style Guide - Instituto Oswaldo Cruz (IOC) - Versão Velha

Este documento resume os padrões de estilo identificados na análise técnica do CSS do portal.

## Paleta de Cores
| Categoria | Hex | Aplicação |
| :--- | :--- | :--- |
| **Brand Primary** | `#114B74` | Links, Títulos Principais, Menus |
| **Brand Secondary** | `#193F62` | Background de Testeira / Cabeçalho |
| **Accent** | `#2D6671` | Títulos de Seções e Notícias |
| **Background** | `#F4F4F4` | Áreas de fundo e seções alternadas |
| **Bordas** | `#D3D3D3` | Divisores e inputs |

## Typography
- **Fonte Principal (UI):** `Montserrat` (100 a 900)
- **Fonte Legado (Marca):** `ioc_fonte` (Museo)
- **Fonte de Apoio:** `Open Sans`

### Escala de Tamanhos
- `H1`: 40px
- `H2`: 32px
- `H3`: 24px
- `Texto`: 18px (line-height: 1.8 para acessibilidade)

## Padrões de Interface (Tokens)
- **Border Radius:** 10px em imagens e blocos; 16px em cards de notícia.
- **Transições:** `0.2s ease-in-out` para todos os estados de hover.
- **Grid:** Layout centralizado com largura máxima de `1140px`.
